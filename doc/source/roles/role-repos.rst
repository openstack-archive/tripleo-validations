=====
repos
=====

An Ansible role to check the correctness of current repositories.

Requirements
------------

This role could be used before/after an Undercloud or an Overcloud has been
deployed.

Role Variables
--------------

- None

Dependencies
------------

No dependencies.

Example Playbook
----------------

.. code-block:: yaml

    - hosts: undercloud
      roles:
         - role: repos

    - hosts: overcloud
      roles:
         - role: repos
